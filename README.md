# Larabook

A social network inspired by _Facebook_ built with PHP **Laravel**.

> Laravel + Facebook = Larabook 😄

<br />

# Features

-   User authentication
    -   Sign up ✔️
    -   Log in ✔️
    -   Password recovery ✔️
-   Posts
    -   CRUD
    -   Like/unlike posts
    -   Comment
-   Group
    -   CRUD
    -   Join group
    -   Send invitation

<br />

# Tech stack

-   Frontend: Blade, Bootstrap, VueJS
-   Backend: PHP Laravel, MySQL
