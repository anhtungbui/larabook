# Larabook

A social network inspired by _Facebook_ built with PHP **Laravel**.

> Laravel + Facebook = Larabook 😄

<br />

# Features

-   User authentication
    -   Sign up ✔️
    -   Log in ✔️
    -   Password recovery ✔️
-   Profile
    -   Update details ✔️
    -   Add friends
    -   Follow/unfollow ✔️
-   Posts
    -   CRUD ✔️
    -   Like/unlike
    -   Comment ✔️
-   Notifications (via web & emails)
    -   New follower ✔️
-   Search
    -   By name ✔️
-   Group
    -   CRUD
    -   Join group
    -   Send invitation

<br />

# Tech stack

-   Frontend: Blade, Bootstrap, VueJS
-   Backend: PHP Laravel, MySQL
